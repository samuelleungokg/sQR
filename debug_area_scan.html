🐛 Area Scan Debug Page

📋 How to Debug Area Scan:

  1. Open Chrome DevTools (F12 → Console tab)
  2. Click the extension icon and select "🎯 Select Area Scan"
  3. Watch console logs for messages starting with 🎯, 📸, ✅, ❌
  4. Draw selection around a QR code below
  5. Check what happens - popup should appear or toast should show

🎯 Test QR Codes for Area Selection:

Large QR Code
"https://www.google.com"

Medium QR Code
"Hello World"

Small QR Code
"Test123"

🔍 Console Log Guide:
  • 🎯 Starting area selection mode... - Area selection activated
  • 🎯 Area selected: {x, y, width, height} - Selection coordinates
  • 📸 Capturing visible tab... - Screen capture started
  • ✅ Screen captured, cropping... - Capture successful
  • 🎉 QR code found: [data] - QR detected!
  • ❌ No QR code detected - Nothing found

🔧 Expected Results:

✅ When QR code is selected:

  • Toast: "✅ QR code detected!"
  • Popup appears with QR content after 1.5 seconds
  • Console shows: "🎉 QR code found: [data]"

❌ When no QR code in selection:

  • Toast: "❌ No QR code detected in selected area"
  • Console shows: "🎯 No QR codes found in selected area"
